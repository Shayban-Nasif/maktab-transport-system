# RS Transport Solution - Iqra International School Tokyo

A comprehensive school transportation management system with real-time tracking, push notifications, and multi-role access.

## Features

- **Multi-role System**: Admin, Driver, and Parent portals
- **Real-time Tracking**: Live ETA updates based on actual pickup times
- **Push Notifications**: Parents get instant updates (Android/Desktop)
- **Route Management**: Assign drivers for AM/PM sessions
- **Student Management**: Enroll students, set stop orders and travel times
- **Live Status**: Parents can track their child's journey

## Tech Stack

- HTML5/CSS3 with modern responsive design
- Vanilla JavaScript (ES6+)
- Firebase (Authentication, Firestore, Cloud Functions, FCM)
- GitHub Pages for hosting

## Setup Instructions

1. Clone the repository
2. Configure Firebase in `js/config/firebase.js`
3. Deploy Cloud Functions: `cd functions && npm install && firebase deploy --only functions`
4. Enable GitHub Pages in repository settings
